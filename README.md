# Portfolio — Edwine Mathieu

Portfolio statique (HTML / CSS / JS, sans framework) : Accueil, CV et Projets.

## Aperçu local

Ouvrir `index.html` dans un navigateur, ou lancer un petit serveur local :

```
npx serve .
```

## Ajouter les photos

Voir [`images/README.md`](images/README.md) — dépose les fichiers avec les noms attendus, ils remplacent automatiquement les placeholders.

## Déploiement GitHub Pages

1. Repo Settings → Pages
2. Source : `Deploy from a branch` → branche `main`, dossier `/ (root)`
3. Le site est servi à l'URL fournie par GitHub Pages.
